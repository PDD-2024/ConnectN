Accept and handle user input.

Takes in the user input. Depending on the state of the Game, uses it to set one of the players' names, set up a board of the appropriate size, or play a move. If in one of the set up states (getPlayer1Name, getPlayer2Name, or getBoardSize), update to next state

◆ player_turn()

int Game::player_turn ( int column)

Completes a player's turn by putting their piece on the board in a specified column. A piece will be placed in the column on the board, unless the column is out of bounds or already full. The value of playsNext is updated to prepare for the other player's turn.

Parameters
columnThe column where the piece will be placed
Returns
0 for success, -1 for an out-of-bounds column, 1 for a full column
